# 🔒 内网环境使用说明

## 问题描述

您遇到的错误：
```
HTTPSConnectionPool(host='www.bing.com', port=443): Max retries exceeded
[WinError 10013] 以一种访问权限不允许的方式做了一个访问套接字的尝试。
```

这表明您的环境无法访问外部网络，可能是由于：
- 内网环境限制
- 防火墙阻止
- 代理设置问题
- 网络权限限制

## 🎯 解决方案

我已经为您创建了**内网版AI助理**，完全不依赖外网连接！

### 📁 文件说明

- **`ai_assistant_offline.py`** - 内网版主应用
- **`start_offline.sh`** - 内网版启动脚本
- **`requirements_offline.txt`** - 精简版依赖包

### 🚀 启动内网版应用

```bash
# 方法1：使用启动脚本（推荐）
./start_offline.sh

# 方法2：手动启动
source ai_assistant_env/bin/activate
streamlit run ai_assistant_offline.py --server.port 8501 --server.address 0.0.0.0
```
