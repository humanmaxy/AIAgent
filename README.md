# 🤖 AI智能助理

一个功能强大的AI助理应用，集成了网络搜索、数据分析和可视化功能，基于您本地的Ollama服务。

## ✨ 主要功能

- **🤖 AI对话**: 与本地Ollama模型进行智能对话
- **🔍 网络搜索**: 自动搜索相关信息
- **📊 数据分析**: 智能分析搜索结果并生成见解
- **📈 可视化**: 生成图表展示分析结果
- **💻 现代界面**: 基于Streamlit的美观用户界面

## 🚀 快速开始

### 前提条件

1. 确保您的Ollama服务正在运行 (http://172.27.66.166:11900)
2. Python 3.8+ 环境

### 安装和运行

1. **克隆或下载项目文件**

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **启动应用**
   ```bash
   # 方法1: 使用启动脚本
   python run.py
   
   # 方法2: 直接使用streamlit
   streamlit run ai_assistant.py
   ```

4. **访问应用**
   - 打开浏览器访问: http://localhost:8501

## 📋 功能说明

### 🔍 搜索并分析
- 输入您想了解的主题或问题
- 系统会自动搜索相关信息
- AI会分析搜索结果并提供综合见解
- 生成可视化图表展示数据分布

### 💬 直接对话
- 与AI进行直接对话
- 无需网络搜索，基于AI知识库回答

### ⚙️ 配置选项
- **模型选择**: 选择不同的Ollama模型
- **搜索结果数量**: 控制搜索结果的数量
- **详细内容获取**: 是否获取网页的详细内容

## 🛠️ 技术架构

- **前端界面**: Streamlit
- **AI模型**: Ollama (本地部署)
- **网络搜索**: Bing搜索引擎
- **数据处理**: Pandas + BeautifulSoup
- **可视化**: Plotly

## 📊 分析功能

### 搜索结果分析
- 来源网站分布统计
- 关键主题提取
- 内容摘要生成

### 可视化图表
- 搜索结果来源饼图
- 关键词相关度条形图
- 实时数据统计

## 🔧 自定义配置

如需修改Ollama服务器地址，请编辑 `ai_assistant.py` 文件中的 `OllamaClient` 初始化参数：

```python
st.session_state.ollama_client = OllamaClient("http://your-ollama-server:port")
```
