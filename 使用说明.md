# 🤖 AI智能助理 - 使用说明

## 🚀 快速启动

### 方法1：使用Shell脚本（推荐）
```bash
./start.sh
```

### 方法2：使用Python脚本
```bash
python3 run.py
```

### 方法3：手动启动
```bash
# 激活虚拟环境
source ai_assistant_env/bin/activate

# 启动应用
streamlit run ai_assistant.py --server.port 8501 --server.address 0.0.0.0
```

## 📱 访问应用

启动后在浏览器中打开：**http://localhost:8501**

## 🔧 配置信息

- **Ollama服务器**: http://172.27.66.166:11900
- **Web界面端口**: 8501
- **Python版本**: 3.13.3

## 🎯 主要功能

### 1. 🔍 智能搜索分析
- 输入您想了解的主题
- 点击"🔍 搜索并分析"
- AI会自动搜索网络信息并进行深度分析

### 2. 💬 直接对话
- 与AI进行直接交流
- 基于Ollama本地模型
- 支持多种模型选择

### 3. 📊 数据可视化
- 自动生成搜索结果分析图表
- 来源网站分布饼图
- 关键词相关度条形图

## ⚙️ 侧边栏设置

- **模型选择**: 选择不同的Ollama模型
- **搜索结果数量**: 3-10个结果可调
- **详细内容获取**: 是否获取网页完整内容
- **清空历史**: 重置对话记录

## 🔍 功能详解

### 网络搜索引擎
- 使用Bing搜索API
- 智能内容提取和清理
- 支持中文和英文搜索

### 数据分析功能
- 关键词提取和频次分析
- 信息源可信度评估
- 内容摘要生成

### AI模型集成
- 本地Ollama服务调用
- 支持多种开源大语言模型
- 可自定义模型参数

## 🚨 注意事项

1. **网络连接**: 确保网络畅通以使用搜索功能
2. **Ollama服务**: 确保Ollama服务正常运行
3. **浏览器兼容**: 推荐使用Chrome、Firefox等现代浏览器
4. **内存使用**: 大量搜索和分析可能占用较多内存

## 🛠️ 故障排除

### 应用无法启动
```bash
# 检查虚拟环境
ls ai_assistant_env/

# 重新安装依赖
source ai_assistant_env/bin/activate
pip install -r requirements.txt
```

### Ollama连接失败
- 检查Ollama服务是否运行
- 确认服务器地址和端口正确
- 尝试在浏览器访问: http://172.27.66.166:11900

### 搜索功能异常
- 检查网络连接
- 可能遇到反爬虫限制，稍后重试
- 尝试使用不同的搜索关键词

## 📞 技术支持

如遇到问题，请检查：
1. Python环境和依赖包是否正确安装
2. Ollama服务是否正常运行
3. 网络连接是否稳定
4. 浏览器控制台是否有错误信息
